--- layout: default title: Local Extrema ---

To find the local extrema of a function you first need to find the critical points. This is done by setting each component of the gradient to zero.

  • If D > 0 and f_{xx}(a, b) > 0, then f(a, b) is a local minimum
  • if D > 0 and f_{xx}(a, b) < 0, then f(a, b) is a local maximum
  • if D < 0, then (a, b) is a saddle point
